This is MWC/RTR X11R5 for COHERENT 4.2, installation command is:

/etc/install CohX /dev/fva0 2

75dpi.tgz	Full set of font files from a X11R5 CD-ROM, including a shell
100dpi.tgz	script to build the font directories in /tmp. After building
misc.tgz	move /tmp/75dpi to /usr/X11/lib/fonts etc. If you install the
		100dpi fonts add them to Fontpath in /usr/X11/lib/Xconfig.
