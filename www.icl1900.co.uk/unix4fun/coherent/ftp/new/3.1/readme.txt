cmd.tar		- On COHERENT 3.1 some commands don't exist, but can
		  be created from 3.2 sources. COHERENT 3.x sources are
		  located in /usr/src/sys, create this directory, copy
		  tar archive to there and extract. Install the executables
		  with make; make install; make clean in the sub-directories
		  in /usr/src/sys/cmd.
